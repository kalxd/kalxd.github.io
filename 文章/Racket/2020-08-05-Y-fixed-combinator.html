Y不动点组合子流水帐 - 2020年08月05日

标签：Racket, 程序语言

Y组合子

Y组合子是 lambda 演算中的一个概念，是任意函数的不动点，在函数式编程中主要作用是提供一种匿名函数的递归方式。

Y组合子表达式：


λf. (λx. f(x x))(λx. f(x x))

递归匿名函数

这里有个递归版本的求和函数：

(define (sum n)
  (if (= n 0)
      n
      (+ n (sum (- n 1)))))

(sum 10)
; 10

大道至简，一目了然。sum内调用了自身，因为它已具名，那么匿名函数又将如何递归呢？

失败的尝试

我们移除sum之后会如何？

(λ (n)
  (if (= n 0)
      n
      (+ n (?f (- n 1)))))

连函数定义都不完整了，?f未定义，看来这样写不行。

穷人的尝试

任何问题都可以通过增加一个间接层来解决。

我们增加一个参数：

(λ (f n)
  (if (= n 0)
      n
      (+ n (f f (- n 1)))))

函数定义完成，f即自身。但如何调用呢？它连名字都没有。我们暂时赋予它一个名字，试着调用：

(define P (λ (f n)
  (if (= n 0)
      n
      (+ n (f f (- n 1))))))
(P P 10)
; 55

我们可以看到P和f并不关联，换句话说，P完全可以被替换掉：

((λ (f n)
  (if (= n 0)
      n
      (+ n (f f (- n 1)))))
 (λ (f n)
  (if (= n 0)
      n
      (+ n (f f (- n 1)))))
 5)
 ; 15

Y组合子定义

定义推导已经很多版本了1。这里直接给出定义：

严格版：

#lang racket/base

(define (Y f)
  (let ([g (λ (x) (x x))]
        [h (λ (y) (f (λ (x) ((y y) x))))])
    (g h)))

(define sum
  (Y (λ (f)
       (λ (n)
         (if (= n 0)
             n
             (+ n (f (- n 1))))))))
(sum 5)
; 15

惰性版：

#lang lazy

(define Y (λ (f) ((λ (x) (f (x x))) (λ (x) (f (x x))))))

(define sum
  (Y (λ (f)
       (λ (n)
         (if (= n 0)
             n
             (+ n (f (- n 1))))))))

(sum 10)
; 10
